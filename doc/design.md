# 用户人群

- 未续报学员家长
- 咨询学员
- 咨询学员家长
- 老师
- 招生人员
- 管理员

# 角色关系

- 游客：登录自动注册，但未有任何角色分配，也未绑定任何学生的帐号。
- 学生：只有档案数据，不能直接登录，可以绑定数据给家长。
- 家长：绑定学生数据后成为家长帐号，可以管理学生信息及课程操作。
- 老师：需要管理分配角色，管理授课和学生管理相关功能。
   - 授课老师
   - 班主任
   - 辅导老师
- 招生：需要管理分配角色，管理接待数据和活动相关功能。
- 管理：分配权限角色给相关帐号。
